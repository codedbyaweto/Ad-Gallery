# Ad Gallery

A responsive web gallery that showcases curated display ads from platforms like Facebook, Google, Twitter, LinkedIn, Quora, and Reddit, with a modern grid layout and light/dark mode support.

## Overview

**Ad Gallery** is a front-end web project designed to present a curated collection of digital advertisements in a clean, card-based layout. It focuses on responsive design, accessibility, and visual consistency using modern web technologies.

This project is ideal for practicing UI layout, theme switching, and front-end structuring.

## Features

- Responsive grid layout  
- Light and dark mode toggle  
- Image-based ad cards  
- Category tags for each ad  
- Custom SVG logo and icons  
- Lightweight and fast performance
